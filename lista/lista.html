Em HTML, as tags "UL" e "OL" são usadas para criar listas. A tag "UL" representa uma lista não ordenada, onde os itens aparecem com marcadores (geralmente pontos) e a ordem dos elementos não é importante. Já a tag "OL" cria uma lista ordenada, onde os itens são numerados, indicando que a sequência tem significado. Em ambas, os itens da lista são definidos com a tag "LI". Essas listas são úteis para organizar informações de forma clara e estruturada em uma página web.
