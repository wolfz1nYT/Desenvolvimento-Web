# Desenvolvimento-Web
Repositório para os trabalhos de Desenvolvimento Web - 2025 

Repositório destinado aos trabalhos e exercícios desenvolvidos durante o curso de Desenvolvimento de Sistemas.

## Conteúdo

Neste repositório estão armazenadas atividades relacionadas a:

* Estrutura básica de páginas HTML
* Títulos e parágrafos
* Listas ordenadas e não ordenadas
* Tabelas
* Formulários
* Links e imagens
* Organização de conteúdo

## Objetivo

Praticar os fundamentos do HTML e compreender a estrutura de páginas web.

## Observações

Os projetos presentes neste repositório foram desenvolvidos para fins acadêmicos e de aprendizado.
